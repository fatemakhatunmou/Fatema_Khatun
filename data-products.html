Data Products

Kiel Institute Africa Monitor


The Africamonitor is a macroeconomic information plattform for Africa. It provides research briefs (data stories), custom visualizations, and centralized access to the leading economic indicators on African economies. The underlying database is global in scope (covering all 195 countries) and updated on a monthly basis. It contains >1000 series collected from 16 public sources (mostly IMF and World Bank databases). Nearly all selected series have high coverage of African economies (50 on average) and other countries. API packages for R and Python allow free high-speed access to the entire database.

South Africa Macroeconomic Database and Nowcast


The South Africa Macroeconomic Database (SAMADB) is an open relational database with >10,000 macroeconomic time series for South Africa, obtained from the South African Reserve Bank (SARB) and Statistics South Africa (STATSSA) and updated on a weekly basis via EconData and automated scraping of the SARB and STATSSA websites. It is accessible through API packages for R, Python and Julia. The SA nowcast is an automated weekly nowcast of GDP and Unemployment (updated every Friday) based on a mixed-frequency dynamic factor model and 57 core macroeconomic time series that receive real-time updates through EconData. The model is able to attribute changes in the nowcast to new data releases.

Uganda Macro Data Portal and Time Series Database


An online data portal and time series database (UGATSDB) providing access to more than 12000 macroeconomic, financial and social time series on Uganda provided by the Bank of Uganda, MoFPED, UBOS, the World Bank, the IMF, and the ILO. Data are provided at the highest available frequency (daily, monthly, quarterly or annual) but can be filtered and aggregated in the portal. Data can be downloaded in various standard formats (Excel, CSV, TSV, STATA, R, SPSS and SAS). In addition, an R API package permits fast and easy access to tidy macroeconomic time series data in R.

Web-Application | Open Source Version | R API

Uganda Macroeconomic Monitor and GDP Nowcast


An interactive dashboard that tracks macroeconomic developments in Uganda using key indicators compiled by national bodies and available at high frequencies. The dashboard also presents a nowcast of quarterly GDP based on key monthly indicators.

Uganda Census Mapper


A comprehensive interactive visualization of the 2014 Census, displaying more than 200 census variables at different spatial resolutions, from macro-regions down to the parish level. Additional indices resembling the dimension-indices in the Human Development Index (HDI) are computed from census variables to display a census-HDI. The full census is also available for download in the Macro Data Portal.

Web-Application | Open Source Version

Website for Uganda Macroeconomic Policy Department incl. Semi-Automated Monthly Economic Performance Reports


Homepage for the Macroeconomic Policy Department (MEPD) in the Ministry of Finance, Planning and Economic Development (MoFPED) of Uganda. It provides access to national macroeconomic data, economic monitoring tools, and the departments regular macroeconomic performance reports. The monthly report has partly automated tables and graphics (thanks to UGATSDB) and is publishable to both HTML and PDF, facilitating its rapid production and consumption.

BRAC Data Explorer


The BRAC Data Explorer is a web application for quick exploratory analysis and geo-visualization of survey data (as frequently collected by BRAC Research). The apps permits upload of data in various formats, and exploratory statistical analysis (summary statistics, histograms, correlations, PCA, clustering, and model selection - guided by the free book 'An Introduction to Statistical Learning with Applications in R'). The geo-visualization module provides interactive spatial visualizations powered by the Leaflet JavaScript library.
